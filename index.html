What is Shell Scripting


A shell script is the terminology for a computer program that can be run by the Unix shell. (BASH CSH KSH)
- What you can run shell scripts, you can run in the terminal.

Automated execution of code in an operating system shell.
Other references depend on operating systems:
- Batch Files (MSDos-Win95)
- Command Procedures (OpenVMS)

Runs on non-extension files.

Uses


Program command line segments in chunks, creating shortcuts for shell commands.
Allows a convenient method to execute system commands.
- Specifications and post-processing properties of the shell all apply naturally while still behaving as a normal Unix command.

Advantages


The commands & syntax are exactly the same as what you would input into a terminal.
Writing a shell script for a program is generally quicker than equivalent code for other languages.
Easy file selection, debugging, etc.

Disadvantages


Be careful!
Commands such as rm, cp, mv differ from other commands with just a single letter.
With typos, you can *accidentally* break your system.
Likewise, * / vs */
Shell commands execute slowly.
Compatibility with different platforms.

            {
                local num1=$1 - $1 indicates the 1st positional parameter

Conditionals


            

            sum=19
            if [ $((9 + 10)) -eq $sum ] - There needs to be a space after the left bracket and before the right bracket
            then
                echo "9 + 10 = $sum"
            elif [ $((9 + 10)) -lt $sum ] - Other inequality symbols include ne,gt,ge(not equal, greater than, greater than or equal to)
            then
                echo "9 + 10 < $sum"
            else
                echo "9 + 10 > $sum"
            fi

            if ((sum == 19)) - Another format for conditionals
            then
                echo "Your number equals 19"
            fi

            if (( ((9 < 10)) && ((11 > 10)) )) - In the previous format, it would be -o or -a
            then
                echo "10 is greater than 9 and less than 11"
            fi
            

            
Output

            

            
            9 + 10 = 19
            Your number equals 19
            10 is greater than 9 and less than 11

Arrays- Bash arrays can only have one dimension


            

            nums=(1.2 1.3 1.4 1.5) - Instantiate arrays with elements inside ()
            echo "${nums[0]}" - Prints the first element in the array
            nums[4]=1.6 -  Adds number to array
            nums+=(1.7 1.8) -  Adds a group to the array
            echo "${nums[@]}" -  Prints all the values in the array
            echo "${- nums[@]}" - Gives the length of the array
            unset 'nums[0]' -  Deletes element
            echo "${nums[@]}"
            unset nums -  Deletes array
            echo "${nums[@]}"
            

            
Output

            

            
            1.2
            1.2 1.3 1.4 1.5 1.6 1.7 1.8
            7
            1.3 1.4 1.5 1.6 1.7 1.8
            (There is supposed to be a blank here)
